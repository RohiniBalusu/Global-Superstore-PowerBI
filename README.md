# 📊 Global Superstore Sales Analysis Dashboard - Power BI Project

## 📌 Project Overview

This project analyzes the Global Superstore dataset using Power BI to provide actionable business insights related to sales performance, profitability, customer behavior, and regional trends. The dashboard enables stakeholders to monitor key performance indicators and identify areas requiring attention through interactive visualizations and drill-through analysis.

---

## 🎯 Project Objectives

* Analyze overall sales and profit performance.
* Identify top-performing and underperforming products.
* Evaluate customer purchasing behavior.
* Analyze regional business performance.
* Perform root cause analysis for sales and profit decline.
* Create interactive dashboards for business decision-making.

### 5. Root Cause Analysis (Drill-Through)

This dashboard provides detailed investigation capabilities for:

#### Sales Decline Analysis

Drill Path:

* Region → State → City → Category → Product

#### Profit Decline Analysis

Drill Path:

* Category → Sub-Category → Product → Discount

#### Customer Analysis

Drill Path:

* Segment → Customer → Last Purchase Date → Recency

#### Regional Analysis

Drill Path:

* Region → State → City → Category
